The electronic appendix attached to this thesis contains the complete source code, data processing pipelines, and analytical scripts used to achieve the results presented in this work.

The repository is structured into two main parts:
\begin{itemize}
    \item \textbf{\texttt{source/}}: The core Python codebase, including modules for web scraping, NLP processing, predictive modeling, and data analysis.
    \item \textbf{\texttt{praca/}}: The \LaTeX{} source files, bibliography, and figures used to compile this thesis document.
\end{itemize}

For detailed information regarding the complete directory structure, required dependencies, and step-by-step instructions on how to replicate the environment and run the code, please refer to the \textbf{\texttt{README}} file included in the root directory of the appendix.

The latest version of the source code is also publicly available on GitHub at: \\ \url{https://github.com/kubovotny/bachelor-thesis}.
