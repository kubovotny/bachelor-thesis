\markboth{Appendix A}{Appendix A}

The electronic appendix is officially submitted as a compressed archive (\texttt{.zip}) through the university information system (AIS2) and contains the complete source code and materials used in this thesis.

The archive is structured into two main directories:
\begin{itemize}
    \item \textbf{\texttt{source/}} -- The core Python codebase, including modules for web scraping, LLM-based NLP processing, data integration, predictive modeling, and statistical analysis.
    \item \textbf{\texttt{praca/}} -- The \LaTeX{} source files, bibliography database, and figures used to compile this document.
\end{itemize}

For detailed information regarding environment configuration (\texttt{requirements.txt}) and step-by-step replication instructions, please refer to the \textbf{\texttt{README.md}} file in the root directory.

The latest version of the source code is also publicly available on GitHub at: \\ 
\url{https://github.com/kubovotny/bachelor-thesis}.
